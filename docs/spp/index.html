Southwest Power Pool Grid Constraint Classifier

Identifying where the grid is congested and where DERs can help
Built by WattCarbon | Data: Southwest Power Pool 2025

What This Tool Does

  • Pulls 10,818,302 hourly LMP data points from Southwest Power Pool (2025), decomposing each into congestion, energy, and loss components
  • Classifies all 1327 Southwest Power Pool pricing zones as transmission-constrained, generation-constrained, both, or unconstrained using weighted multi-factor scoring
  • Scrapes 30 data center records from interconnection queues and maps them to Southwest Power Pool zones, identifying growth pressure areas
  • Generates DER deployment recommendations per zone, aligned with WattCarbon WEATS asset categories (dispatchable, consistent, variable)
